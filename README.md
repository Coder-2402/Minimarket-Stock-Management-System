# Mimimarket Stock Management System
Contains the final practical exam project for the Data Structures & Algorthms (DSA) course.

## About
This program implements C++ to simplify inventory management in a minimarket. Using a Binary Search Tree (BST) data structure, an in-order search function, and an insertion sort function, this system enables efficient and organized management of item data (ID, name, stock).

## Main Features
- Adding, updating, and deleting items
- Searching for items by ID or name
- Fast and structured data access
